# Bestie 2.0 💕

Heather's forever AI companion.

## Setup on Replit

1. Import this repo into Replit
2. Click the 🔒 **Secrets** tool in left sidebar
3. Add two secrets:
   - `OPENROUTER_KEY` = your OpenRouter API key
   - `FISH_AUDIO_KEY` = your Fish Audio API key
4. Click **Run** (green button)
5. Open the Webview URL on your iPhone 17 Pro Max

## Voice Toggle (Money Saver)
Keep voice OFF by default. Toggle ON only when you want Bestie to speak.
When toggled OFF, **zero** API calls are made to Fish Audio.
